## Dev

1. Clonar el archivo .env.template a .env
2. Configurar las variables de entorno en el archivo .env
3. Ejecutar el comando `npm install`
4. Levantar las bases de datos con el comando
   ```bash
   docker-compose up -d
   ```
5. Ejecutar
   ```bash
   npm run dev
   ```
